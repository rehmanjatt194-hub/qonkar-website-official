● OVERVIEW

Redesign and Redevelopment of the QUEST University Website

This project involved the complete redesign and redevelopment of the official website of QUEST (Quaid-e-Awam University of Engineering, Science & Technology), Nawabshah. The previous website faced several usability and design issues that hindered user engagement and accessibility. My goal was to transform the platform into a modern, responsive, and user-focused website that better represents the university’s identity and serves the needs of students, faculty, and visitors.

● PROBLEMS

The Problem for Users

Outdated Interface & Poor Performance
The original website suffered from multiple issues that negatively impacted user experience and site performance:

  • ✖ Outdated and visually unappealing interface
  • ✖ Lack of responsiveness across different screen sizes
  • ✖ Inconsistent design elements
  • ✖ Poor emphasis on clear calls to action (CTAs)
  • ✖ Wasted space leading to inefficient layout
  • ✖ Small, poorly arranged images affecting visual appeal
Previous UI

The primary objective of this project was to redesign and redevelop the website to deliver a modern, user-friendly, and engaging platform. The redesign aimed to achieve the following goals:

  • ✔ Increase website traffic and attract more users
  • ✔ Improve click-through rates on the Recent News section
  • ✔ Inconsistent design elements
  • ✔ Enhance user engagement and dwell time
  • ✔ Reduce the website’s bounce rate
